## Database

数据来源：中国外汇交易中心

### 2. NDF市场（NEX报价）

### 3. CHN市场（NEX报价 / 即期汇率定盘价）
数据来源：Wind

## 数据处理
### 1.AR—GARCG模型拟合
CNY，CNH和NDF的数据不满足平稳条件，做一阶差分得到收益率数据，后做ADF test，满足平稳条件  
做Ljung-Box test，可以拒绝独立性假设，因此观察ACF和PACF图，根据AIC准则，选用最优AR(m)模型，对残差做LB test，依然不通过  
观察残差和残差平方图，发现有异方差性，因此采用AR-GARCH模型做联合估计，得到标准化残差，
